ERPAssist Installation Instructions
====================================

QUICK INSTALL (3 Commands):
---------------------------

1. Extract this ZIP file to your bench apps directory:
   
   unzip erpassist.zip -d ~/frappe-bench/apps/

2. Get the app (this adds it to apps.txt):
   
   cd ~/frappe-bench
   bench get-app ./apps/erpassist

3. Install on your site:
   
   bench --site your-site.local install-app erpassist
   bench --site your-site.local migrate
   bench restart


ALTERNATIVE METHOD (If bench get-app doesn't work):
---------------------------------------------------

1. Extract ZIP to apps folder:
   
   unzip erpassist.zip -d ~/frappe-bench/apps/

2. Manually add to apps.txt:
   
   cd ~/frappe-bench
   echo "erpassist" >> sites/apps.txt

3. Install dependencies:
   
   pip install -e ./apps/erpassist

4. Install on site:
   
   bench --site your-site.local install-app erpassist
   bench --site your-site.local migrate
   bench restart


CONFIGURE:
----------

After installation:

1. Login to your ERPNext site
2. Search for "ERPAssist Settings" 
3. Enter your OpenAI API Key
4. Save
5. Refresh browser (Ctrl+Shift+R)
6. Click chat icon in navbar
7. Start chatting!


TROUBLESHOOTING:
----------------

If you get "App not in apps.txt":
  echo "erpassist" >> ~/frappe-bench/sites/apps.txt

If you get "Module not found":
  cd ~/frappe-bench/apps/erpassist
  pip install -e .

If UI doesn't appear:
  bench build --app erpassist
  bench clear-cache
  bench restart
